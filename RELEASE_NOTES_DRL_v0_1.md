# Overview

# Key Architectural Principle

The central architectural contribution of this release is the explicit distinction between observations and evidence.

```text
Observation ≠ Evidence
```

Raw observations become laboratory evidence only after provenance has been established.

Scientific interpretation is intentionally postponed until later analytical layers.

# Scientific Contribution

Difference Retention Laboratory introduces an experimental architecture in which provenance is preserved as a first-class architectural invariant.

Instead of processing anonymous measurements, the laboratory operates on evidence objects with explicit origin and traceability.

This architecture separates:

* observation;
* evidence;
* environment adaptation;
* experiment execution;
* scientific interpretation.
